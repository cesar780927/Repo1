# 📂 Procesamiento de Archivos y Texto en Python

Este repositorio contiene mis soluciones y prácticas correspondientes al programa de especialización en Python de la **Universidad de Michigan** en Coursera. 

El objetivo principal de este proyecto es demostrar el dominio de los fundamentos de Python aplicados a la lectura, análisis y extracción de información desde archivos de texto plano (`.txt`).

---

## 🛠️ Tecnologías y Conceptos Practicados

* **Lectura eficiente de archivos:** Uso de la estructura `with open(...)` para garantizar el cierre seguro de los recursos en memoria.
* **Manipulación de strings y listas:** Segmentación de texto con `.split()`, conteo selectivo de caracteres y filtrado de palabras mediante ciclos `for` y condicionales `if`.
* **Manejo de rutas relativas:** Implementación de la librería estándar `pathlib` para asegurar la portabilidad del código entre diferentes sistemas operativos (Windows/Linux).
* **Control de versiones:** Uso de comandos esenciales de **Git** y vinculación con **GitHub** para la gestión del portafolio local y remoto.
